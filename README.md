Metadata Extractor and Correlation Tool

Description This project is a Python-based Metadata Extractor and Correlation Tool developed to automatically extract meaningful metadata from structured data files such as CSV and text files. The extracted metadata is cleaned, organized, and correlated to identify basic relationships between metadata attributes.

The tool reduces manual effort in metadata analysis and improves data understanding for academic and basic data analytics applications.

Features

Extracts metadata from CSV and text files
Performs data cleaning and preprocessing
Removes duplicate and irrelevant metadata
Correlates metadata attributes
Generates structured output in tabular format
Technologies Used

Python
Pandas Library
CSV / Text File Handling
Windows / Linux / macOS
How to Run the Project

Clone the repository: git clone https://github.com/your-username/metadata-extractor.git

Navigate to the project folder: cd metadata-extractor

Install the required dependency: pip install pandas

Run the application: python code/main.py

Output

The tool generates structured metadata after processing the input dataset.
Cleaned and correlated metadata is stored in CSV format.
Output includes:
File name and file type
Metadata attributes
Correlation details
Output File Location: output/result.csv

Conclusion The Metadata Extractor and Correlation Tool provides a simple and efficient solution for extracting and organising metadata. Its modular design and ease of use make it suitable for students and beginners in data analytics and data science.
